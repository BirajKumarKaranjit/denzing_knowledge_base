# NBA Analytics Knowledge Base

Welcome to the NBA Analytics Knowledge Base, a comprehensive resource for understanding and leveraging data in the realm of professional basketball. This knowledge base is designed to provide detailed insights into database structures, business rules, SQL best practices, and response guidelines, all tailored to the unique demands of NBA analytics.

## Sections

### ddl
This section contains the table schemas and column definitions for all NBA database tables. It provides a detailed blueprint of the data architecture, ensuring clarity and consistency in data management and retrieval.

### business_rules
Explore the NBA domain metrics, KPI definitions, and calculation formulas in this section. It offers a deep dive into the methodologies used to quantify player performance, team efficiency, and other critical basketball analytics.

### sql_guidelines
The SQL Guidelines section offers best practices for crafting efficient and accurate SQL queries. It includes query patterns, join conventions, and highlights dialect-specific gotchas to avoid common pitfalls in data querying.

### response_guidelines
This section outlines the standards for output formatting and response quality. It ensures that data outputs are presented in a clear, consistent, and actionable manner, facilitating better decision-making and communication.
