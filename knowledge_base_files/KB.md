# NBA Analytics Knowledge Base

Welcome to the NBA Analytics Knowledge Base, your comprehensive resource for understanding and leveraging basketball data. This knowledge base is designed to provide detailed insights into the structure, rules, and best practices for analyzing NBA data, ensuring accurate and meaningful interpretations.

## Sections

### ddl
This section contains the table schemas and column definitions for all NBA database tables. It provides a detailed overview of the data structure, helping you understand how data is organized and stored.

### business_rules
Explore the NBA domain metrics, KPI definitions, and calculation formulas in this section. It outlines the key performance indicators and metrics used in NBA analytics, along with the formulas for calculating them.

### sql_guidelines
The SQL Guidelines section offers SQL query patterns, join conventions, and dialect-specific gotchas. It serves as a guide for writing efficient and accurate SQL queries tailored to NBA data analysis.

### response_guidelines
This section provides output formatting and response quality guidelines. It ensures that the results of your analyses are presented clearly and effectively, maintaining high standards of communication.
