# Denzing — Product Overview

## What Is Denzing?

Denzing is an **AI-powered Answer Engine for enterprise data**. It replaces static dashboards and
back-and-forth analyst requests with conversational AI Agents that answer business questions in
real time — grounded in your actual data, with transparent and traceable reasoning.

> "Answers, Not Reports."

The core insight: dashboards were never built for curiosity. Denzing Agents behave like a network
of collaborating data analysts, understanding both the **structure** of your data (schema) and its
**meaning** (business context), so users can ask natural questions without writing SQL.

---

## The Problem Denzing Solves

Traditional analytics is a relay race:

- Executives ask questions → analysts build queries → dashboards are created → follow-up questions restart the cycle.
- Building a dashboard takes **4–6 weeks** on average and costs **$10,000–$100,000**.
- Dashboards cannot handle follow-up questions or explore new angles.

Denzing replaces this with a single intelligent Agent that responds in chat, instantly.

---

## Core Agent Capabilities

| Capability | Description |
|---|---|
| **Insight & Analysis** | Synthesizes and summarises findings — doesn't just present raw numbers |
| **Intent Clarification** | Asks follow-up questions when the user's intent is ambiguous |
| **Transparent Reasoning** | Users can see inside the Agent's thinking — not a black box |
| **Insight Sharing** | Share findings and invite others to ask follow-up questions |
| **Evolving Knowledge Base** | Agent gets smarter and more personalised the more it is used |
| **Collaboration** | Humans and Agents work in the same conversation thread |
| **Full Decision Spectrum** | Handles quick operational checks and deep multi-step analysis |

---

## How Agents Are Deployed (3 Steps)

1. **Point** — Connect securely to your existing data sources.
2. **Train** — The Agent learns your schema, business metrics, and how your organisation talks about data.
3. **Explore** — Users chat with the Agent and share insights across the organisation.

## This NBA Agent — Context

The **NBA AI Agent** is a demonstration instance of the Denzing platform, built on the NBA
dataset. It showcases Denzing's ability to:

- Answer natural-language basketball questions (player stats, team performance, streaks, comparisons)
- Generate SQL against a structured sports data schema automatically
- Provide contextual analysis alongside raw numbers
- Handle multi-turn conversational follow-ups

It operates on NBA regular season and playoff data and defaults to the **most recent available
season** unless the user specifies otherwise. The agent does not have access to real-time game
scores or live data — all data is historical and loaded into the warehouse.
